### Ways to Contribute

1. **Share What Works** â€” Found a technique that improved your AI visibility? Document it.
2. **Add Case Studies** â€” Real-world examples help everyone learn.
3. **Fix & Improve** â€” See outdated info or errors? Submit a PR.
4. **Cite Sources** â€” Help build a research-backed knowledge base.
5. **Ask Questions** â€” Open discussions that help others learn.

### How to Contribute

```bash
# Fork and clone the repository
git clone https://github.com/YOUR_USERNAME/aeo.git

# Install dependencies
npm install

# Start the development server
npm run dev

# Create a branch for your changes
git checkout -b feature/your-contribution

# Make your changes and commit
git add .
git commit -m "Add: description of your contribution"

# Push and open a PR
git push origin feature/your-contribution
```

### Content Guidelines

- **Cite your sources** â€” Link to research, case studies, or data that supports claims
- **Be specific** â€” Share actual numbers, examples, and implementation details
- **Stay current** â€” This field changes rapidly; note when information was last verified
- **Avoid promotion** â€” This is educational content, not marketing material
- **Share learnings** â€” Even failed experiments teach us something
